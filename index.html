Sobre o Projeto

Esse é o Projeto Ginius! Baseado no jogo Genius (ou Simon, em inglês), esse projeto foi desenvolvido para praticar conhecimentos em HTML, CSS e JavaScript.

Para iniciar a partida, selecione a caixa "ligar" e pressione o botão "START". Caso você selecione o modo "estrito", você perde o jogo se cometer um erro.

Faça uma sequência de 20 turnos para ganhar!
