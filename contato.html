Sobre mim

Prazer, Gigi! :)

É graças a ele que tive contato com HTML, CSS e JavaScript. Fiz esse jogo para praticar ainda mais, pois é só assim que se aprende!

Espero que você se divirta com o Ginius! (aliás, pegou o trocadilho? risos!)
